今年度からコンピュータ工学Ⅰではエディタとしてマイクロソフトが開発しているVisual Studio Code (以下、VS Codeと表示)を使います。VS Codeは多くのプログラミング言語をサポートしており、Windows、Linux、Macでも使え現在、最も人気の高いエディタの一つです。では、さっそくインストールしましょう。

ダウンロード

  • ここをくりっくしてウェブサイトに飛び、赤枠の部分をクリックする。
  • 以下のウインドウが開くので「保存」をクリックする。
  • 保存したVSCodeUserSetup-x64-1.44.2.exeをダブルクリックする。
  • 「使用許諾契約書の同意」ウインドウが開くので「同意する(A)」にチェックを入れて、「次へ(N)>」をクリックする。
  • 「インストール先の指定」ウインドウが開くので、そのまま「次へ(N)>」をクリックする。
  • 「プログラムグループの指定」ウインドウになるので、そのまま「次へ(N)>」をクリックする。
  • 「追加タスクの選択」ウインドウになるので、全てチェックを入れてから「次へ(N)>」をクリックする。
  • 「インストール準備完了」ウインドウになるので、「次へ(N)>」をクリックするとインストールが始まる。
  • 「インストール状況」ウインドウになり、インストールの状態がわかる。
  • インストールが終わると、次のウインドウになり、「完了(F)」をクリックするとVS Codeが起動する。
  • 次のようにVS Codeが起動したら成功。
  • お疲れ様！

終わり。
