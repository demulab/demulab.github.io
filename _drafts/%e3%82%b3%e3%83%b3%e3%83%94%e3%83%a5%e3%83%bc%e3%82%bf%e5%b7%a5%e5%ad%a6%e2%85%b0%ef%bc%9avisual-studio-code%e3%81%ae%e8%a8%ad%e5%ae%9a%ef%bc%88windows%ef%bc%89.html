Visual Studio Code（以下、VS Codeと略記）のインストールが終わったので次に設定して使いやすくします。

日本語化

  • デスクトップにある「VS Codeアイコン」をダブルクリックして起動する。
  • 「View」をクリックして「Command palette..」を選択する。
  • 「Configure Display Language」を選択。
  • 「Install additional languages ..」を選択。
  • 「Japanese Langeuage Pack V... 日本語」の「Install」をクリック。
  • 右下に「In order to use VS Code in Japanese . VS Code needs to restart」と表示されるので、「Restart Now」をクリック。
  • 下図のように日本語化されたVS Codeが開く。
  • お疲れ様！
