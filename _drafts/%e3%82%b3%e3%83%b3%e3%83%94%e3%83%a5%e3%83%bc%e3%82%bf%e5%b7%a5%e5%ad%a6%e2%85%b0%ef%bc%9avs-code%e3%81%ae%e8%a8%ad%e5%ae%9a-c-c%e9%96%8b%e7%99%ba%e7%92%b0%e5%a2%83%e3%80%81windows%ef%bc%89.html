C言語の授業で使うのでC/C++の開発環境をWindows版のVS Codeにインストール・設定します。コンパイラーはGccを使います。

環　境

  • Windows 10 Education 64 bit, バージョン：2004, OSビルド：190241.207

Microsoft C/C++拡張機能のインストール

  • VS Codeを起動して開く。
  • 左にあるサイドバーの拡張機能をクリックする。
  • １の検索窓に「C/C++」と入力すると、２に「C/C++拡張機能」が表示されるので「インストール」をクリックする。
  • C/C++拡張機能がインストールされる。

Mingw-w64のインストール

  • Mingw-w64はWindows上でLinuxで標準となっているgccコンパイラーをサポートしているインストールする。ここをクリックしてダウンロードする。
  • ダウンロードしたmingw-w64-install.exeをダブルクリックしてインストーラーを起動する。
  • 「この不明な発行元からのアプリがデバイスに変更を加えることを強化しますか？」と表示されるので「はい」をクリックする。
  • 「Welcome to the MinGW-W64 Installation!」の画面になるので「Next >」をクリックする。
  • 「Settings」の画面になるの以下のように設定して「Next >」をクリックする。デフォルトだと32ビット版のi686なのでWindows10が32ビット版以外（ほとんどの学生）はx86_64を選択する。
    • Version:8.1.0
    • Architecture:x86_64    (Windows10が32ビット版の場合はi686を選択）
    • Threads: posix
    • Exception: seh
    • Build revision:0
  • 「Installation folder」の画面になるのでDestination folderを以下に変更（ディレクトリ名に空白があるとトラブルの元なので初めの"\Program Files"を消す）して、「Next >」をクリックする。
    • C:\mingw-w64\x86_64-8.1.0-posix-seh-rt_v6-rev0
  • 「Installing Files」の画面になるのでしばらく待つ。私の環境（光回線）では数分かかった。緑色のバーが右端までいき「Click Next to continue the installation.」と表示されたら「Next >」をクリックする。
  • 以下の画面になったらMinGW-W64のインストールは終了。「Finish」をクリックしてインストール作業をやめる。
