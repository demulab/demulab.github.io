Deep Learning用に購入したNVIDIA GTX 1080Ii搭載パソコンの環境構築メモ。

環境

  • パソコン
    • CPU: Core i7-7700
    • Motherboard: Asus PRIME H270-PLUS
    • Memory: 8GB
    • Video card: Asus TURBO-GTX1080Ti-11G
    • Case: SST-T J04B-E
    • Power supply: SST-ST75F-P 750W

 

  • xubunut 16.04.2
  • Kernel 4.8.0-36-generic
  • gcc/g++ 5.4.0

作業

  •  nvidiaドライバインストール
    • sudo add-apt-repository ppa:graphics-drivers/ppa
    • sudo apt-get update
    • sudo apt-get install nvidia-381
  •  NVIDIAのウェブサイトからCuda8.0のインストール
    • cuda_repo-ubuntu1604-8-0-local-ga2_8.0.61-1_amd64.debをダウンロード
    • sudo ./cuda_7.5.18_linux.run
    • いろいろ聞かれるが、NVIDIAドライバはすでにインストールしているので入れない。OpenGLを入れるとXが立ち上がらなくなるといった記事があったので入れない。後はデフォルトのまま。
    • .bahrcに以下を追加したCUDAのパスを通す
      • export CUDA_HOME=/usr/local/cuda
      • export　LD_LIBRARY_PATH=${LD_LIBRARY_PATH}:/usr/local/cuda/lib64
  • NVIDIAのウェブからcuDNNのインストール
    • tar xvzf cudnn-7.5-linux-x64-v5.1.tgz
    • cd cuda
    • sudo cp -a lib64/lib*   /usr/local/cuda/lib64/
    • sudo cp include/cudnn.h  /usr/local/cuda/include/
