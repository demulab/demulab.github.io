本記事は金沢工業大学ロボティクス学科２年次後学期PD実践の講義用です。

本講義では、金沢工業大学基礎実技教育課程の西川教授から強力なサポートを受けてETロボコンをロボティクス学科向けにアレンジしたロボコンをテーマとして、組込みプログラミングの基礎を楽しく学びます。なお、以下の方法は西川教授の資料を参考にしています。

本記事は、EV3の画面に文字や絵を表示することができるLCDのAPIを学びます。

  • API
    •  小さいサイズのフォントを設定する
      •  ev3_lcd_set_font(EV3_FONT_SMALL);
    • 指定した位置（x=0,y=10）に”HELLO”を表示する
      • ev3_lcd_draw_string("HELLO", 0, 10);
    •  指定した位置（x=20,y=30）を始点として、幅40、高さ50の矩形を描き、黒色を塗る
      •  ev3_lcd_fill_rect(20, 30, 40, 50, EV3_LCD_BLACK);
    • LCD画面全体をクリアする
      • ev3_lcd_fill_rect(0, 0, EV3_LCD_WIDTH, EV3_LCD_HEIGHT, EV3_LCD_WHITE);
  • サンプルプログラム
    • /lcd/app.c：LCDにシステム時間とカウンタ、バッテリー残量を表示する。
  • 演習
    • サンプルプログラムを読んでどのように動作するかを考えてからEV3にアップロードし、実際の動作と比較する。
    •  サンプルプログラムを改造し、以下の機能を実現する。
      「countが100になったら、LCDに“Game Over!”と表示する」
  • 参考
    •  LCD画面の幅は178、高さは128である。
    •  LCD画面の左上隅が、座標（x=0,y=0）である。
    •  フォントサイズは以下の通り。
      • EV3_FONT_SMALL 小さいサイズのフォント：幅6×高さ8
      • EV3_FONT_MEDIUM 普通サイズのフォント：幅10×高さ16
