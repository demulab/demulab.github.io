ROS Indigoの標準ではgazebo2が入るが古いのでgazebo4にバージョンアップする。

  • sudo apt-get install ros-indigo-viz
  •  rosのgazebo2関連パッケージをアンインストールする
    • apt-get remove ros-indigo-gazebo-*
  • gazebo4をインストールする。
    • wget -O /tmp/gazebo4_install.sh http://osrf-distributions.s3.amazonaws.com/gazebo/gazebo4_install.sh; sudo sh /tmp/gazebo4_install.sh
  • rosのgazebo4関連パッケージをインストールする。apt-getだと依存関係をうまく解決できないので、aptitudeを使う。aptitudeがない場合はapt-get install aptitudeでインストールする。
    • sudo aptitude install ros-indigo-gazebo4-ros
