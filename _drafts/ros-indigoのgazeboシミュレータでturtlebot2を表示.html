このメモはkendemuさんのこの記事を参考にしました。kendemuさんありがとう。

      • Gazebo4のインストール：ここからDebian用のファイルをダウンロードし、以下のコマンドでインストールする。

＄　sh ~/download/gazebo4_install.sh

  • ROS作業用のディレクトリを作る。既にある場合はそれを使えば以下の作業は不要。ただ、既存のturtlebot_siulatorとは別のディレクトリにする。
    • $ mkdir -p  ~/catkin_ws/src
    • $ cd ~/catkin_ws/src
    • $ catkin_init_workspace
    • $ cd  ~/catkin_ws
    • $ catkin_make
    • $ source  devel/setup.bash
  • ソースインストール作業
    • $ cd ~/catkin_ws/src
    • $ git clone https://github.com/turtlebot/turtlebot_simulator.git
    • $ rosinstall ~/catkin_ws/src/turtlebot_simulator /opt/ros/indigo
    • $ echo "source ~/catkin_ws/src/turtlebot_simulator/setup.bash" >> ~/.bashrc
    •  以下のコマンドで必要なパッケージをインストールする
      • $ sudo apt-get install ros-indigo-gazebo4-ros
      • $ sudo apt-get install ros-indigo-gazebo4-plugins
      • $ sudo apt-get install ros-indigo-gazebo4-ros-control
      • $ sudo apt-get installros-indigo-gazebo4-ros-pkgs
      • $ sudo apt-get install ros-indigo-gazebo-concert
      • $ sudo apt-get install ros-indigo-gazebo-msgs
