The State Machine Compilerを使った時のメモ。

  • The State machine CompilerのウェブサイトからSmcSrc_6_5_0.zipを~/srcへダウンロード。
  • cd src
  • unzip SmcSrc_6_5_0.zip
  • C++のサンプルを実行
    • cd ~/src/smc/examples/C++/EX1
    • make checkstring
    • ./checkstring 00000
