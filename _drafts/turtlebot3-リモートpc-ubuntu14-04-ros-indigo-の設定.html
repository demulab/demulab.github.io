TurtleBot3 Offcial Wikiでは、Ubuntu 16.04とROS Kinetic KameでのリモートPCの設定について書いてあるが、手持ちのノートPCはUbuntu14.04 + ROS Indigoなので設定してみる。

依存パッケージのインストール

  • cd ~/catkin_ws/src/
  • git clone https://github.com/ROBOTIS-GIT/turtlebot3_msgs.git
  • git clone https://github.com/ROBOTIS-GIT/turtlebot3.git
  • cd ~/catkin_ws && catkin_make

package 'openni-dev' not foundと出てcatkin_makeが失敗する。
