Kinect V2でObject Recognition Kitchenの使用に関するメモ。Object Recognition Kitchenのサイトを参考に実施。Ubuntu14.04だと問題なくObject Recognition Kitchenはaptでインストールできるが、Ubuntu16.04はtabletopなど一部のパッケージがないのでソースからビルドした。

インストール

  • sudo apt-get update
  • sudo apt-get upgrade
  • sudo apt-get install ros-kinetic-object-recognition-*
  • sudo apt-get install ros-kinetic-moveit-core
  • metslibのインストール
  • sudo apt-get install couchapp
  •  tabletopのバイナリがないのでソースからビルド
    • cd ~/catkin_ws/src
    • git clone https://github.com/wg-perception/tabletop.git
    • cd ~/catkin_ws
    • catkin_make
    • topic名が違うので~/catkin_ws/src/tabletop/confのdetection.object.ros.orkとdetection.table.ros.orkを次のように書き換える。

      parameters:
      #rgb_frame_id:'/kinect_head_rgb_optical_frame'
      #rgb_image_topic: '/kinect_head/rgb/image_rect_color'
      #rgb_camera_info: '/kinect_head/rgb/camera_info'
      #depth_image_topic: '/kinect_head/depth_registered/image_rect'
      #depth_camera_info: '/kinect_head/depth_registered/camera_info'
      rgb_frame_id: kinect2_rgb_optical_frame
      rgb_image_topic: /kinect2/hd/image_color_rect
      rgb_camera_info: /kinect2/hd/camera_info
      depth_image_topic: /kinect2/hd/image_depth_rect
      depth_camera_info: /kinect2/hd/camera_info
