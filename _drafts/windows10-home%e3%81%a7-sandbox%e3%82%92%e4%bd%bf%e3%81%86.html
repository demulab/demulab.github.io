Windows Sandboxはシステムを怪しいソフトから守るための軽量な仮想マシンであり、Windows10 ProやWindows 10 Enterpriseなら使用できる。ただ、Windows Homeを使っている方も多いと思う。僕のPCもWindows Homeだが、Windows SandboxをFolding@Homeで試してみたかった。いろいろ調べてみるとパッチを当てると使えるようなので試してみた。なお、Windowsのオフィシャルではないので、試される方は自己責任となる。この記事は以下の参考サイトを要約しているだけ。

参考サイト

  • Enable Windows Sandbox Feature in Windows 10 Home Edition by Windows TenForums

ダウンロード

  • sandbox-install.zipをDeskmodderからダウンロードする。ここをクリック。
  • sandbox-install.zipを解凍すると、sandbox-insallディレクトリができて、その中にSandbox Installer.batファイルが生成される。そのファイルをダブルクリックする。
  • インストーラーが起動して作業が行われ、終了すると以下の文が表示されるので"Y"をキー入力してEnterキーを押す。コンピュータが再起動する。
    • Do you want to restart the computer now? (Y/N)

確　認

  • WindowsキーとRキーを同時に押す。「ファイル名を指定して実行」ウインドウが開くので以下を入力して、"OK"をクリックする。
    • OptionalFeatures.exe
  • 「Windowsの機能」ウインドウが開くので、以下のように「Windows Sandbox」にチェックが入っていることを確認する。これでOK！
