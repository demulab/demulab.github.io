Folding@Homeの名を語りフィシングメールでマルウェアを送り付けるあり問題になったようだ。WindowsのSandboxで実行するとマルウェア被害から感染を防げる。MicrosoftがSandboxでFolding@homeを実行するPowerShellのスクリプトを公開したので試してみた。

環　境

  • Windows 10 Pro, Enterprise Insider build 18362以降
  • BIOSで仮想化機能(Virtualization Technology)を有効にする。

参考サイト

  • Folding In Sandbox by microsoft

ダウンロード

  • 以下のWindows Sandbox Utilitesをダウンロードして解凍する。この例では、C:\Users\ユーザ名にダウンロードして解凍する。
    • https://github.com/microsoft/Windows-Sandbox-Utilities/archive/master.zip

実　行

  • 管理者権限でコマンドプロンプトまたはパワーシェルを起動して次のコマンドを実行する。
    • cd C:\Users\ユーザ名\Windows-Sandbox-Utilities-master\Folding In Sandbox
    • Powershell.exe -ExecutionPolicy Bypass -File .\install_folding_sandbox_on_host.ps1
  • ユーザ名を指定したい場合は次のコマンドを実行する。
    • cd C:\Users\ユーザ名\Windows-Sandbox-Utilities-master\Folding In Sandbox
    • Powershell.exe -ExecutionPolicy Bypass -File .\install_folding_sandbox_on_host.ps1 -username <your username>

  • Sandboxが自動的に起動するが、以下のような「Error 0x80072746. 既存の接続はリモートホストにより強制的に切断されました」と表示される場合。

  • 設定→アプリ→プログラムと機能→Windows機能の有効化または無効化で次の項目にチェックが入っているか確認する。
    • Hyper-V
    • Window Sandbox
  • 入っていなければ、チェックを入れて「OK」をクリックする。
