仮想シミュレーション環境「箱庭」の使い方を説明する。

  • cfgフォルダの設置。これは初回だけ実施すればよい。
    • Ubuntuのターミナル（端末）を起動して以下のコマンドを実行する。
      • $ cd ~/src/ev3rt-athrill-ARMv7-A
      • $ mkdir cfg/cfg
      • $ cp cfg/cfg-linux-64  cfg/cfg/cfg
      • $ chmod +x  cfg/cfg/cfg
  • EV3ロボット制御プログラムのビルド
    • 端末で以下のコマンドを実行してビルドする。
      • $ cd ~/src/ev3rt-athrill-ARMv7-A/sdk/OBJ1.1
      • $ make clean
      • $ make "ATHRILL_BUILD_TARGET = ubuntu18"
    • ビルドが成功するとaspという実行可能ファイルが生成される。
  • Unityシミュレータの起動
    • Unityの▶ボタン（下図参照）を押して、シミュレーションモードにする。

  • Athrillの起動
    • 端末で以下のコマンドを実行してathrillを起動する。
      • $ cd ~/src/ev3rt-athrill-ARMv7-A/sdk/OBJ1.1
      • UDP版の場合
        • $ athrill2 -c1 -t -1 -m memory.txt -d device_config.txt ./asp
      • MMAP版の場合
        • $ athrill2 -c1 -t -1 -m memory_mmap.txt -d device_config_mmap.txt ./asp
