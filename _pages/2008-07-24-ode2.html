ODEの開発環境並びにインストールについて説明します．ご自分の環境に合わせて以下のリンクをたどってください．なお，Code::BlocksはオープンソースでありマルチプラットフォームのC/C++統合開発環境です．日本ではあまりメジャーではありませんが，海外でユーザが多いです．ODEではVisual C++と並んでInstall方法にその説明があります．Windowsだけではなく，Linuxも使わなければならない多くのロボット開発者にはお勧めです．

  • 開発環境のインストール
    • Windows 10
      • Code::Blocks 20.03 (2020-06-21更新)
    • Linux (Ubuntu 18.04)
      • 以下のコマンドでインストール可能。ただし、バージョンは16.01。
        • $ sudo apt install codeblocks
      • 日本語化
        • 次のサイトからcodeblocks_ja_17.12_105.tar.gzをダウンロードして、展開してできたlocaleディレクトリを/usr/share/codeblocksディレクトリ下にコピーする。日本語の設定は次のサイトを参照。
          • Code::Blocks 日本語化
  • ODEのインストール
    • Windows 10
      • Code::Blocks 20.03 (2020-06-21更新)
    • Linux (Ubuntu18.04)
      • Configure (2020-06-21更新)
  • 自作ODEプログラムのビルド法
    • Windows10
      • Code::Blocks 20.03 (2020-06-21更新)
    • Linux (Ubuntu18.04)
      • Makefile (2020-06-25更新)
