今年のつくばチャレンジはステレオカメラを使う予定なのでその作業メモ。

  • CORIANDERのインストール
    • ここからcoriander-2.0.0.tar.gzをダウンロード
    • tar xvzf  coriander-2.0.0.tar.gz
    • ./configure
      • エラーが出たら必要なパッケージをインストール。私の場合は以下をインストール
      • sudo  apt-get install libgnomeui-dev
    • make
    • sudo make install
    • sudo  /sbin/ldconfig
    • カメラをつないでcorianderを起動. デフォルトではスーパーユーザでないとvideoデバイスにアクセスできない。
      • sudo  coriander
