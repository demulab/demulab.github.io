SoftbankのPepperをROSで動かすメモ。今回はTeleop。ノートパソコンのキーボードを使い、Pepperを動かしてみた。簡単に動かせた。さすがは、ROS。

  • teleop_twist_keyboardのインストール
    • cd ~/catkin_ws/src
    • git clone https://github.com/ros-teleop/teleop_twist_keyboard.git
  • 以下のコマンドをそれぞれ別の端末で実行する
    • roscore
    • roslaunch pepper_bringup pepper_full_py.launch nao_ip:=192.168.0.2 （このIPは操作するペッパーのものに変更）
    • rosrun teleop_twist_keyboard teleop_twist_keyboard.py
  • 最後のコマンドrusrun teleop...を実行した端末に表示してあるキーでPepperを動かすことができる。注意する点としては、Pepperの後ろ下部にある電源プラグのふたを完全に閉じないとPepperが動かないこと。下の写真のように少し浮いていて30分ぐらい無駄にした。一番下の写真のようにしっかり閉めましょう。
