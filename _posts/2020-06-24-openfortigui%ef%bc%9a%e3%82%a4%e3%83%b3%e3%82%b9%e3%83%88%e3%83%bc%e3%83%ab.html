LinuxでのVPN接続アプリを探している方も多いと思います。
特に、WindowsでFortiClientを使っている方に、Fortigate VPN-ハードウェアに接続するオープンソースのLinuxクライントアプリOpneFortiVPNとそのGUIであるOpenFortiGUI を紹介します。

参考リンク

  • OpenFortiGUI, Bits and Bytes
  • openfortivpn

環　境

  • Ubuntu18.04

準　備

  • Qt5とその関連パッケージが必要なのでインストールする。
    • $ sudo apt install qtbase5-dev qttools5-dev-tools qt5-default qt5keychain-dev

インストール

  • OpenFortiVpn
    • $ sudo apt install openfortivpn
  •  OpenFortiGUI
    •  バイナリのインストール。以下のdebファイルをインストールする。
      • OpenFortiGUI 0.8.2 64bit
    •  ソースからのインストール。次のコマンドを実行する。
      • $ cd ~/src
      • $ git clone https://github.com/theinvisible/openfortigui.git
      • $ cd openfortigui
      • $ git submodule init
      • $ git submodule update
      • $ qmake
      • $ make
      • $ cd openfortigui

実　行

  • $ ./openfortigui

設　定

  • ファイル→セットアップウイザード→OpenFortiGUI初期設定（AES鍵の設定）→完了

  • 追加→VPN→VPNの設定
    • 名前、VPNサーバー、ユーザ、パスワードなどを設定する

使い方

  • OpenFortiGUIを起動する。
    • openfortigui &
  • 接続したいVPNをクリックして選択肢、「接続」をクリックする。

  • ログイン画面になるので、ユーザー名とパスワードを入力して「送信」をクリックする。

 

  • 接続されると、状態が「接続済み」に変わる。
