Blender2.8に組み込まれているPython3にモジュールを追加する方法。標準ではあまり入っていないので必要な作業。以下のサイトを参考にした。

  • 参考サイト
    • https://blender.stackexchange.com/questions/56011/how-to-install-pip-for-blenders-bundled-python
  • pipのインストール
    • BlenderのPython用にpipをインストールする必要があるので、既にpipがインストールされていても作業を行う必要がある。
    • 端末を開いてBlenderがインストールされているディレクトリに移動する。この例ではソースからインストールしている。aptやsnapでバイナリインストールした場合は、ディレクトリを適宜読み替える。
      • $ cd ~/src/blender/2.82/python/lib/python3.7/bin
    • 以下のリンクからget-pip.pyをダウンロードする。
      • pip documentation
        • $ curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
    • get-pip.pyの実行
      • $ cd ~/src/blender/2.82/python/lib/python3.7/bin
      • $ ./python3.7m  ./get-pip.py
  • モジュールのインストール
    • moduleという名前をインストールする場合。インストールするmoduleがnumpyの場合は、moduleをnumpyに置き換える。
      • $ cd ~/src/blender/2.82/python/lib/python3.7/bin
      • $ ./python3.7m  pip install module
    • とりあえず、以下のモジュールはよく使うのでインストールしておく
      • $ cd ~/src/blender/2.82/python/lib/python3.7/bin
      • $ ./python3.7m pip install numpy
      • $ ./python3.7m pip install pillow

これでBlenderでいろいろできる。
