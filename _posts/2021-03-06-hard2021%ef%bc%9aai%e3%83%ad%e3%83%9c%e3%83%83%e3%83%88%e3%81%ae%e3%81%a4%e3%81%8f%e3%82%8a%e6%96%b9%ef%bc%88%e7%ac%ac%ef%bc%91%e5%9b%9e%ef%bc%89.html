HARD2021スプリングワークショップ「第１回AIロボットのつくり方」のサポートページ。HARD2021は日本ロボット学会インテリジェントホームロボティクス研究専門委員会主催の「知能ホームロボティクス入門講習会2021」の基礎編として実施します。

  • スライド
    • 公開用HARDスプリング1-210306
  • 次回までのハンズオン
    • Linux（はじめての方）
      • 研究開発やサーバーなどの用途では広く使われているOS（オペレーティングシステム）。知能ロボットでは標準的に使われています。
      •  勉強方法
        • オンライン教材：ざっと観て概要を知ろう。無料レッスンだけでOK
          • Linux入門編 (全5レッスン)、Paizaラーニング　(無料レッスンだけでよい）
        • 教科書（無料)
          • Linux標準教科書、LPI-Japan　(次回までに1章と3章、後は必要に応じて)
    • 開発環境のインストール
      • 次回のワークショップで使いますので必ずインストールを済ませておいてください。
      • 注意：必ず、Ubuntuのバージョンは18.04、ROSのバージョンはMelodicにしてください。他のバージョンでは本ワークショップのソフトウェアは動きません。
      • 対象機器
        • PC（Windows10 または Ubuntu 18.04)　または Raspberry Pi4 (RAM  8GB)
      • シミュレータのみの場合
        • HARD2021：USB Linux (Ubuntu18.04)の作り方
        • Ubuntu18.04: ROS Melodicのインストール
      •  ルンバを用意できる場合
        • ノートPCで動かす場合
          • 「シミュレータのみの場合」と同じものをインストールしてください。
        • Raspberry Pi4（RAM8GB版)で動かす場合
          • RAM 8GB版
            • Raspberry Pi4 (8GB RAM): Xubuntu18.04 のインストールと設定
            • Raspberry Pi4 (8GB RAM)：ROS Melodicのインストール
            • Raspberry Pi4 (8GB RAM): ROS MelodicでRoombaを動かそう！
    • シミュレータのインストール
      • 次リンクのシミュレータもインストールしておいてください。ハンズオンですぐ使います。
        • HARD2021: ミュレータでRoombaを動かそう！
  • ワークショップに必要な知識とスキルの勉強法
    •  ROS
      • 以下のROS Tutorialを３月中に終える
        • ROS Tutorial (日本語）
        • ROS Tutorial (最新版、英語）
    •  タッチタイプ
      • 習得に要する期間：一ヶ月〜二ヶ月
      •  プログラミングする前に必要な技能です。マスターしていない方は１日30分程度とレーニングしてできるだけ早く身に付けよう。いろいろ練習するソフトはあり好きなものを使ってください。ここでは、参考までに以下を紹介します。私の妻と息子は特打でマスターしました。ゲーム感覚で楽しめます。
        • 無料：美佳のタイプトレーナ
        • 有料：特打 (1980円)
    • Python
      • 習得に要する期間：数カ月〜半年（他の言語をマスターしていれば早い）
      • AIやデータ処理で標準的なコンピュータ言語です。知能ロボットには必須です。米国のコンピュータサイエンス系学科では、初年度にPython言語を学んでいる大学が多いです。HARD2021ではPython言語を使います。
      • 勉強方法
        • オンライン教材：ざっと観て概要を知ろう。無料レッスンだけでOK
          • Python3入門編（全１１レッスン）Paizaラーニング
        • 教科書：７章の関数までは８月中に終え、それ以降は必要に応じて勉強しよう。
          • やさしいPython、高橋麻奈著、SBクリエイティブ, 2828円
        • オンラインジャッジ：問題を解くことにより理解が深まります。以下のリンクのIntroduction to Programming Iを解ければ初級は卒業です。こういうことが好きなら競技プログラミングの世界もあります。
          • AIZU ONLINE JUDGE, Introduction to Programming 1
  • 質問と回答
    • ワークショップにはノートPCが必ず必要ですか？
      • Raspberry Pi4でも可能です。ノートPCかRaspberry Pi4のどちらかが必要です。
    • Macでもできますか？
      • すみませんが、Macを持っていないので対応できません。
    • Raspberry Pi4 RAM 2GBでは、このサイトの説明どおり実施してもインストールできません。
      • HARD2021では、Raspberry Pi4 8GBを対象とします。2GBだとメモリが少なく実用的でないと思います。
